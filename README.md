# 🤖 LinkedIn Automation Bot Dashboard

**Professional web interface** for automated LinkedIn post & comment liking. Works with **Personal Profiles** or **Company Pages** using Google Sheets.

[![Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://share.streamlit.io)

## ✨ Features
- 🔐 **Secure login** - Anyone enters their own LinkedIn + Google credentials
- 👥 **Multi-user** - Works for 3rd parties instantly
- ⚙️ **Personal or Company mode** with automatic profile switching
- 📊 **Live logs + metrics** (Posts processed, success rate)
- 📱 **Google Sheets integration** - Status updates automatically
- 🛡️ **Stealth Selenium** - Human-like delays & mouse movements
